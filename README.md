# PeekProof Landing Page

Modern, sleek landing page for PeekProof - the app that catches phone snoopers.

## Features

- 🎨 Dark theme with purple accents (matching the app)
- 📱 Responsive design
- ⚡ Static export for fast hosting
- 🔒 Privacy Policy & Terms pages

## Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Connect to Vercel
3. Deploy!

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```
